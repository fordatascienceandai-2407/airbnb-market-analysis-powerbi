# 🏠 Airbnb Market Analysis using Power BI

## 📌 Project Overview

This project analyzes Airbnb listings from **Chicago** and **New Orleans** using **Power BI**. The objective of the project is to understand market trends, pricing behavior, host performance, customer engagement, and listing availability through interactive dashboards.

The project demonstrates end-to-end data analysis including:

* Data Cleaning
* Data Transformation
* Data Modeling
* DAX Calculations
* Dashboard Design
* Business Analysis
* Storytelling

---

# 🎯 Project Objectives

The main objectives of this project were:

* Compare Airbnb markets across cities.
* Analyze pricing patterns and room type distribution.
* Evaluate host performance.
* Study customer engagement.
* Analyze listing availability.
* Generate actionable business insights.

---

# 🛠️ Tools Used

* Power BI Desktop
* Power Query
* DAX
* Data Modeling
* Data Visualization

---

# 📊 Data Preparation

The following data preparation steps were performed:

* Imported Chicago and New Orleans Airbnb datasets.
* Performed data profiling.
* Validated data types.
* Cleaned missing values.
* Added City column.
* Appended both datasets into a single master table.
* Created DAX measures and calculations.

# 💡 Key Insights

* Chicago has more Airbnb listings than New Orleans.
* Entire home/apartment is the dominant room type.
* Hotel rooms represent the premium accommodation segment.
* Certain neighborhoods command significantly higher prices.
* Professional hosts manage a substantial number of listings.
* Customer engagement varies across room types.

---

# 📌 Business Recommendations

* Hosts should invest in high-demand property types.
* Premium neighborhoods can support optimized pricing strategies.
* Availability and minimum stay policies should be monitored to maximize occupancy.
* Smaller markets may provide future expansion opportunities.
